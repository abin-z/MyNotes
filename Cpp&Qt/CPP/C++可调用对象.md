# C++中的可调用对象

在 C++ 中，可调用对象是指那些可以通过 `operator()` 被调用的对象。C++ 提供了多种方式来创建和使用可调用对象.

以下是 C++ 中常见的几种可调用对象:

1. **普通函数(Function)**
2. **函数指针（Function Pointer）**
3. **函数对象（Functor）**
4. **Lambda 表达式**
5. **成员函数指针（Member Function Pointer）**
6. **std::function**
7. **std::packaged_task**
8. **std::bind（绑定函数参数）**
9. **std::invoke**
10. **std::reference_wrapper**
